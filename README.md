# 🔮 Telegram-бот «Пророцтво Тараса»

Цей бот — це інтерактивна міні-гра, яка дарує користувачу натхнення у вигляді випадкової цитати-поради на кожен день. Це простий, але цікавий спосіб додати трохи мотивації у повсякденне життя.

## 🚀 Основний функціонал

* **Привітання:** Користувач отримує тепле привітання («Ну привіт, козаче!») при запуску.
* **Інтерактивна клавіатура:** Керування здійснюється за допомогою зручних кнопок у нижній частині екрана.
* **Генерація пророцтв:** При натисканні кнопки «Надай пророцтво ✨» бот видає випадкову мотивуючу фразу.
* **Захист від помилок:** Бот підказує використати команду `/start`, якщо користувач вводить некоректний текст.

---

## 🛠 Технології

Бот реалізований на мові **Python** з використанням бібліотек для роботи з Telegram API (наприклад, `aiogram` або `telebot`).

---

## 📸 Як це виглядає

| Команда | Опис |
| --- | --- |
| `/start` | Запуск бота та виклик клавіатури |
| **Надай пророцтво** | Отримання випадкової цитати |
| **Завершити роботу** | Закриття меню або деактивація клавіатури |

---

## 📖 Інструкція з використання

1. Натисніть кнопку **"Привіт! Почати! 👋"** для ініціалізації.
2. Виберіть **"Надай пророцтво ✨"**, щоб отримати свою пораду на сьогодні.
3. Якщо ви захочете приховати кнопки, натисніть **"Завершити роботу ❌"**.

---

## 🔧 Встановлення та запуск (для розробника)

1. Клонуйте репозиторій:
```bash
git clone https://github.com/your-username/taras-prophecy-bot.git

```


2. Встановіть необхідні залежності:
```bash
pip install -r requirements.txt

```


3. Створіть файл `.env` та додайте туди ваш токен від BotFather:
```env
TOKEN=ваш_токен_тут

```


4. Запустіть бота:
```bash
python main.py

```
